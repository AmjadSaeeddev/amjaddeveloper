How I solved the client's problem?

I chose the HTML5 for structure, CSS3 for design, Jquery & Javascript for functionilty, PHP for Back-end, SQL FOR Database, Boostrape for repsonsiveniess and lightweight and SASS for reduce and extensibility the Code, Underscore.js for custom theme development and Metabox for custom fields.

Why HTML5 ?

  • Cost effective Multi-Platform Development.
  • Good page ranking.
  • Consistency across multiple browsers.
  • Better mobile access to Business Intelligence.
  • A better user experience.

Why CSS3 ?

  • Easier to maintain and update.
  • Greater consistency in design.
  • More formatting options.
  • Lightweight code.
  • Search engine optimization benefits.

Why JAVASCRIPT?

  • JavaScript is a client side language.
  • JavaScript is comparatively fast for the end user.
  • Extended functionality to web pages.
  • No compilation needed.
  • Event-Based Programming language.

Why PHP?

  • Extremely Flexible. PHP is highly flexible whether it is during an ongoing project or after completing the project.
  • Easy Integration and Compatibility.
  • Efficient Performance.
  • Cost-Efficient.
  • Gives Web Developer More Control.

Why SASS ?

  • Sass facilitates US to write clean, easy and less CSS in a programming construct..
  • It contains fewer codes so we can write CSS quicker.
  • It is more stable, powerful, and elegant because it is an extension of CSS.
  • It is compatible with all versions of CSS.
  • It is maintainable, themeable, and extendable.

Why bootstrap?

  • A consistent framework that supports major of all browsers and CSS compatibility fixes.
  • Lightweight and customizable.
  • Responsive structures and styles.
  • Good documentation and community support.
  • Fewer Cross browser bugs.

Why JQUERY ?

  • Cross-browser compatibility.
  • Clean and beautiful syntax.
  • Animations and cool effects.
  • Highly extensible.
  • Pages load faster.

Why Underscore.js?

  • Underscore.js is one of the most popular modern JavaScript libraries used for functional programming.
  • Clean and beautiful syntax.
  • Most popular.
  • Extensible.
  • Pages load faster.

Why Metabox?

  • Meta Box is a powerful toolkit for developers to create custom meta boxes and custom fields for any custom post type.
  • lightweight.
  • Most popular.
  • professional.
  • Meta Box is a free Gutenberg and GDPR-compatible WordPress.
